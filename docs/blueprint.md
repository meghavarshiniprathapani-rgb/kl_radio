# **App Name**: KL Radio Hub

## Core Features:

- Listen Live: Provides a live audio stream of KL Radio.
- Song Suggestions: Allows users to submit song suggestions for the radio station to play.
- Announcements Feed: Displays important announcements and updates from KL Radio.
- Member Login: Enables members (wings) to log in to access their respective dashboards.
- Wing Dashboards: Customized dashboards for each wing (RJ, Creative, Technical, PR, Designing, Video Editing, Broadcasting) with relevant tools and information.
- Admin Panel: A comprehensive admin panel for the Station Head to manage content, users, and settings.
- Smart Announcement Generator: Uses a tool to create announcement drafts based on user input parameters, streamlining announcement creation.

## Style Guidelines:

- Primary color: A vibrant red (#E63946) to reflect energy and passion for radio.
- Background color: A very light desaturated red (#F1FAEE) to provide a soft backdrop.
- Accent color: A warm yellow (#F4A261) to highlight interactive elements and call to actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look, is used.
- Use modern and simple icons to represent different sections and actions, ensuring clarity and usability.
- Design a mobile-first, responsive layout that adapts seamlessly to different screen sizes, ensuring a consistent experience across devices.
- Incorporate subtle animations and transitions to enhance user engagement and provide visual feedback.
